3 Use SimpleKMeans clustering to understand visits

This R code chunk performs k-means clustering on the Walmart_visits data using the SimpleKMeans function. It first saves the number of unique TripType levels and removes the TripType variable from the input data. Then, it generates clusters with default parameters. Next, it demonstrates different variations of k-means clustering by changing the initial assignment method, distance function, and the number of clusters. Each models results are displayed to compare the clustering solutions.

# Saving TripType as TripType.levels and removing it from the input data

TripType.levels <- length(unique(Walmart_visits$TripType))
Walmart_visits <- Walmart_visits %>% 
  select(-TripType)

# Generating clusters with default parameters
